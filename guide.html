🚀 Primeiros Passos

1 Carregar Dados de Exemplo

No primeiro acesso, carregue dados de exemplo para explorar o sistema com dados da Fast Digital 360.

Ou abra o Console (F12) e digite:

loadSeedData()
2 Cadastrar Pessoas

Vá em Pessoas → Clique em "+ Nova Pessoa"

Preencha: Nome, Cargo e Salário Mensal

3 Criar Squads (Opcional)

Vá em Squads → Clique em "+ Novo Squad"

Dê um nome e selecione os membros do time

4 Cadastrar Contratos

Vá em Contratos → Clique em "+ Novo Contrato"

Preencha cliente, valor e entregáveis. Atribua um squad ou pessoas específicas.

5 Analisar Resultados

Volte para o Dashboard e veja todas as análises automaticamente calculadas!

  • ✅ Ranking geral de eficiência

🧮 Como os Cálculos Funcionam

ROI por Contrato

ROI = Receita - Custo das pessoas atribuídas

Se um contrato de R$ 10.000 tem um squad de R$ 6.000, o lucro é R$ 4.000 (40% de margem)

Custo por Entregável

Custo/Entrega = Salário Mensal / Total de Entregáveis

Se um designer ganha R$ 4.000 e produz 50 criativos/mês, cada criativo custa R$ 80

ROI de Squad

ROI Squad = Receita Total - Custo Total do Squad

Soma todos os contratos do squad e subtrai a soma dos salários

💡 Dicas de Uso

Use Squads para contratos recorrentes
Atribua squads fixos aos clientes que têm demanda previsível. Isso facilita o cálculo de ROI do time.
Atribua pessoas individuais para projetos pontuais
Para trabalhos específicos, atribua pessoas individuais ao invés de squads completos.
Compare profissionais do mesmo cargo
Vá em Comparações → Selecione um cargo para ver quem é mais eficiente em custo por entrega.
Exporte seus dados regularmente
Em Contratos → Clique em "Exportar Dados" para fazer backup do seu trabalho.

🛠️ Comandos do Console

Abra o Console do navegador (F12) e use estes comandos:

loadSeedData()

Carrega dados de exemplo da Fast Digital 360

clearAllData()

Apaga TODOS os dados (cuidado!)

window.exportContracts()

Exporta dados em JSON (mesmo que o botão na interface)

❓ Suporte

O sistema roda 100% no seu navegador. Seus dados ficam salvos no localStorage.

Dados não aparecem após refresh?
Verifique se o localStorage está habilitado nas configurações do navegador.
Erro ao importar módulos?
Use um servidor HTTP local. Não abra os arquivos diretamente (file://).
